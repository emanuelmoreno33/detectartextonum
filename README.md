# detectar Texto, Numeros y Simbolos

El programa detecta:

- [ ] numeros enteros y decimales
- [ ] numeros telefonicos
- [ ] Texto con signos
- [ ] Ecuaciones
- [ ] Texto
- [ ] Direcciones


ejemplos

valor|aparecera
-----|---------
846267 | numeros
6641234567 | numero telefonico
$ho#% | signos
Juan carlos 7 | texto
100.22.3 | signo
a=b+c | ecuacion
todas las letras Y # excepto los demas signos | texto
664-123-45-67 | numero telefonico
Juan carlos #7 | texto
Juan carlos No.7 | texto
27+3 | ecuacion

Las ladas de estados seran
* Baja california
* Baja california sur
* Sonora
* Sinaloa
* Chihuahua
